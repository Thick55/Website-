HTML5 Markup

HTML5 is the latest version of the Hypertext Markup Language, the code that describes web pages. It is a powerful tool for creating modern, interactive websites.

CSS3 Styling

CSS3 is the latest version of the Cascading Style Sheets language, which is used to style web pages. It allows for more advanced styling options and greater control over the layout and design of a website.

Graphic Design

Graphic design is the art and practice of creating visual content to communicate information and ideas. It involves the use of typography, imagery, color, and layout to create effective designs for print and digital media.
